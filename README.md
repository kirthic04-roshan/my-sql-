# StockDataset SQL Project

## Overview
A SQL project analyzing 20 Indian companies across sectors like IT Services, Banking, FMCG, Energy, and more. The dataset includes MarketCap, CurrentPrice, OpenPrice, ClosePrice, Volume, and Exchange, enabling financial and sectoral insights.

## SQL Functions Used
- **Selection**: `SELECT *`, specific columns  
- **Filtering**: `WHERE`, `AND`, comparison operators  
- **Sorting & Limiting**: `ORDER BY`, `LIMIT`  
- **Aggregates**: `AVG()`, `SUM()`, `COUNT()`, `MAX()`, `MIN()`  
- **Grouping**: `GROUP BY` with aggregates  

## Key Insights
- Sector- and exchange-level analysis  
- Ranking by MarketCap, CurrentPrice, ClosePrice  
- Trends in prices, volumes, and capitalization  

## Learning Outcomes
- Mastery of SQL fundamentals & advanced functions  
- Practical application in **business analytics**  
- Hands-on dataset creation & query writing

# FIFA Player SQL Project

## Overview
This project analyzes the `fifa_player` dataset using SQL queries to explore player performance, market value, and club insights. 
It demonstrates the use of **Window, Aggregate, String, Date/Time, and NULL handling functions** through 25 practice questions with answers.

## Functions Covered
- Window: ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, MIN (PARTITION BY)
- Aggregate: AVG, MAX, COUNT, SUM, GROUP BY
- String: UPPER, LENGTH, CONCAT
- Date/Time: NOW
- NULL Handling: IFNULL, COALESCE
- Clauses: OVER, PARTITION BY, ORDER BY

## Outcomes
- Ranking players by ratings, goals, and market value  
- Aggregating statistics for clubs and nationalities  
- Handling missing values and string manipulations  
- Applying SQL for **sports analytics and business intelligence**

## Usage
Clone the repo, import the dataset, and run queries from `queries.sql` to explore results.
